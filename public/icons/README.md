# Icons structure

- `app/` - app-level placeholders and service icons (`market.webp`, `ban.webp`, `app-icon.png`)
- `currency/` - TON, Stars, Wildcoin and currency variants
- `nav/` - navigation and section icons
- `ui/` - generic UI actions and utility icons (close, copy, arrows, etc.)
- `social/` - Telegram/wallet/support related icons
- `flags/` - locale and language flags
- `decor/` - decorative visual assets
